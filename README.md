# Simple-Food-Adding (FastAPI + React Simple Fullstack Project)

This project demonstrates how to connect a **FastAPI backend** with a **React frontend** from scratch.
The goal of this project is to show how a frontend and backend communicate with each other using APIs.
This is a simple educational project did to learn how modern full-stack applications are structured.

---

## What This Project Does

- Creates a **FastAPI backend API**
- Creates a **React frontend application**
- Connects the frontend to the backend
- Sends requests from React to FastAPI
- Returns responses from FastAPI to React
- Fixes **CORS errors** when communicating between frontend and backend
- Demonstrates a simple full-stack workflow

---

## Technologies Used

| Technology | Purpose |
|-----------|--------|
| FastAPI | Backend API |
| Python | Backend programming language |
| React | Frontend framework |
| JavaScript | Frontend logic |
| CORS Middleware | Allow frontend-backend
